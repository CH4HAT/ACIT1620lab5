CSS Flexbox

CSS Flexbox is a one-dimensional (either horizontal or vertical) layout method that can help us quickly create layouts otherwise difficult to achieve with previous techniques (e.g. positioning) discussed in this course.

First and foremost, Flexbox requires any elements you want to lay out to have a parent element set to display: flex. This parent element could be a div, containing several images; it could be the body, containing the header, main, and footer elements; or any other case where you would like to lay out elements in a row or column.

CSS Grid

CSS Grid is a responsive, two-dimensional (both horizontal and vertical) layout method that allows us to easily create complex layouts for the web.

Similar to CSS Flexbox, CSS Grid requires any elements you want to lay out to have a parent element whose display value is set to grid or inline-grid.

The number of columns and rows in a grid are defined by two properties: grid-template-columns and grid-template-rows

or by the shorthand grid property.

Child elements are placed using grid-column and grid-row
